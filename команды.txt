
когда первый раз заходишь нужно залогиниться
$ git config --local user.name "vetal"
и почту зачем-то прописать
$ git config --local user.email ччччччч@gmail.com

добавить все файлы каталога для сохранения версии это с ключем А
$ git add -A
сохранить версию и добавить комментарий ключ m
$ git commit -a -m "1-й коммит"



за коннектить с гитхабом и дать название коннекту Ncode
$ git remote add Ncode https://github.com/vetalm1/Ncode.git
передать на гитхаб в коннект Ncode текущую версию.(вроде так)
$ git push -u Ncode master
